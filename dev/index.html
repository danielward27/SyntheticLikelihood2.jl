SyntheticLikelihood2.Prior — Type

Struct to hold prior distributions. v is a vector of distributions from Distributions.jl. Univariate and/or multivariate distributions can be used. Each element in the vector is assumed to be independent.

source
SyntheticLikelihood2.cut_at — Method

'Cut' vector at specified indexes, to form a vector of vectors. Note the cut occurs after the index, so [1] would cut between index 1 and 2 in the vector.

source
SyntheticLikelihood2.gaussian_simulator — Method

Simulate a three dimensional Gaussian mean vector θ. The covariance is diagonal, and fixed to σ=0.1. Parameter vector θ is the mean vector of the Gaussian

SyntheticLikelihood2.@loopify — Macro

Convenience macro to define a method for a simulator that allows simulating in batches. i.e. it takes simulator(rng::AbstractRNG, θ::Vector{Float64}) and wraps it in a for loop to get simulator(rng::AbstractRNG, θ::Matrix{Float64}).
